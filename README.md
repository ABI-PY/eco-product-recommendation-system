# 🌱 Eco Score Analyzer (Flask Project)

A simple Flask-based web application that analyzes product sustainability using keyword-based scoring and web scraping.

## 🚀 Features
- 🔍 Analyze product eco score from URL
- 💰 Extract product price (if available)
- 🌱 Eco scoring (keyword-based logic)
- ⚖️ Compare two products
- 🕘 History tracking (SQLite)
- 📊 Graph visualization
- 📥 Export data as CSV
- 🤖 Simple chatbot

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- BeautifulSoup (Web Scraping)
- HTML, CSS, JavaScript
